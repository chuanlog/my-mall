## 项目简介
* 这是scut2025秋网络应用架构设计与开发课程的课程设计
* 项目后端基于开源脚手架项目mall-tiny进行开发，基于springboot+mysql+mybatis-plus的技术栈，脚手架项目链接：[脚手架项目](https://github.com/macrozheng/mall-tiny)。
* 前端项目基于react
## 技术栈一览

| 前端技术栈                                           | 后端技术栈                                                                        |
|-------------------------------------------------|------------------------------------------------------------------------------|
| React（前端框架）、React Router（路由）、Ant Design（UI组件库）、Axios（HTTP请求）、Vite（构建工具）、ESLint（代码规范） | Spring Boot（基础框架）、Spring Security（认证与授权）、MyBatis-Plus（ORM增强）、MySQL（关系型数据库）、Redis（缓存）、JWT（令牌认证）、Knife4j（接口文档）、MinIO（对象存储）、Maven（构建管理） |
